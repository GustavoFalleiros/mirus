Mirus Debugging via Serial

Mirus is capable of simple debugging over serial ports. By default, when running Mirus in QEMU via make kvm, a serial port is opened on a file, /tmp/miruslog. This build-in debugging facilities can read and write from this file. You may interface with it via the following preferred ways.
  • tail -f /tmp/miruslog - tail is a utility with will latch onto the file and monitor it for any updates, and then display them.
  • echo "message" >> /tmp/miruslog - This will append "message" to the file, which will be read by Mirus via the serial driver.
